CapStoneApi2

HTTP Method Endpoint Description
GET / or /HiveHome Returns the HTML5 file for the home page
POST /login Logs in a user
GET /users Retrieves a list of all users
PUT /user/:id Updates a user with the specified ID
POST /register Creates a new user
DELETE /user/:id Deletes a user with the specified ID
GET /products Retrieves a list of all products
GET /product/:id Retrieves a single product with the specified ID
POST /product Adds a new product
PUT /product/:id Updates a product with the specified ID
DELETE /product/:id Deletes a product with the specified ID
